Creating a scale model of the solar system

If you were to create a model of the solar system with accurate sizing and distance relationships, you would simply need to divide all dimensions by a large number and then convert them to units we can work with.

Using the input box below divides the actual sizes of the Earth, the Sun, and the distance between and converts those numbers to yards, inches, and millimeters.

Note: An accurate graphical representation cannot be displayed on a computer monitor. Making the Earth the size of a single pixel would require placing the sun over 9 feet away.
